# NetSecurePro-Identity

**Identité Souveraine Post-Quantique pour l'OT Santé**

Brique identité de **Z-CORE OS v2.6** — 100% Offline • 100% PQC • 100% Souveraine.

## Fonctionnalités
- `did:zcore:` — Décentralized Identifier résilient
- Signatures **Dilithium5** + **Falcon**
- Verifiable Credentials (VC) avec **ZKP** (Zero-Knowledge Proofs)
- Module **GRONDIN v2** — Audit & monitoring offline
- "NON par défaut" au cloud étranger

## Cas d'usage immédiat
